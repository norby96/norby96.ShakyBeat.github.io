PROYECTO FINAL DE ARTE Y TECNOLOGÍA

ShakyBeat Medio de expresión corporal musical

INICIO CÓDIGO FUNCIONAMIENTO IMÁGENES

Este proyecto tiene como propósito fomentar la práctica musical por medio de un dispositivo tecnológico que logre reproducir notas musicales de acuerdo al movimiento de los brazos. Por eso, se tiene como enfoque facilitar el empleo de este instrumento artístico para aquellas personas que no cuentan con alguna extremidad superior.

Para la realización de este proyecto se utilizaron los siguientes materiales:

  • Arduino UNO
  • 2 Sensores de 6 Grados de Libertad (MPU-6050)
  • 1 Bluetooth (HC-05)
  • Cables de conexión
  • 1 Batería de 8V
